While you have something highlighted, use Shift + Alt + UpArrow / DownArrow to copy the selected text to next or previous line.

You can create a div with the class "test" by simply typing div.test

You can also create a div with multiple classes by typing div.test1.test2

Let's say you want to create a div with the class "test" and the id "test", you can do that by typing div.test#test

Or just a "p" with the id "placeholder" is p#placeholder

Ctrl + Enter makes a new line regardless of where the cursor is. This is useful when you want to add a new line without breaking the current line. Ctrl + Shift + Enter makes a new line above the current one.
